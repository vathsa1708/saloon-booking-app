# saloon-booking-app
Clone the repository from the link mentioned to local machine.
Install dependencies: Run npm install to install specified dependencies mentioned on
Package.json file.
Set up Environment Variables: Incase failed to add MongoDB or Nodejs server in
environment variables,make sure to add the path.
Database Set Up : Make sure database is working properly by typing mongod --on cmd.
Better to check every time in cmd whether crud operation is performing or not.
Start the Server : Run the server by using node Index.js in terminal.This will start running
server.
Test the endpoints: Using postman ,provide the necessary inputs to ensure backend is
working properly.
